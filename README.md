# Modern Password Generator (Pygame)

Это графическое приложение на Python для генерации безопасных паролей.

## Особенности
* Длина пароля: 20 символов.
* Использует буквы, цифры и спецсимволы.
* Написано с использованием библиотеки Pygame.

## Как запустить
1. Установите Python.
2. Установите библиотеку Pygame:
   ```bash
   pip install pygame
   ```
3. Запустите основной файл:
   ```bash
   python main.py
   ```
